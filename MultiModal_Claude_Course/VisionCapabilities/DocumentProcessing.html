Document Processing Pipelines

PDF Processing with Claude

Claude can process PDF documents directly, understanding both text content and visual layout including tables, charts, and formatted text.

# Send a PDF directly
with open("report.pdf", "rb") as f:
    pdf_data = base64.standard_b64encode(f.read()).decode("utf-8")

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=4096,
    messages=[{
        "role": "user",
        "content": [
            {"type": "document", "source": {
                "type": "base64",
                "media_type": "application/pdf",
                "data": pdf_data
            }},
            {"type": "text", "text": "Extract all financial figures and summarize key findings."}
        ]
    }]
)

OCR Replacement

Claude's vision capabilities often surpass traditional OCR because it understands context and layout, not just individual characters. It can:

  • Read handwritten text with high accuracy
  • Understand table structures without explicit grid lines
  • Extract data from forms with mixed layouts
  • Handle rotated, skewed, or low-quality images
