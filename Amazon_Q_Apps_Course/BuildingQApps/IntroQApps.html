Introduction to Amazon Q Apps

What Are Amazon Q Apps?

Amazon Q Apps is a feature of Amazon Q Business that lets users create lightweight GenAI applications without writing code. Users can create apps from natural language descriptions or from Q Business conversations.

Key Features

  • Conversation to App: Turn any Q Business conversation into a reusable app
  • Card-Based Builder: Drag-and-drop interface with input/output cards
  • Enterprise Data Access: Apps inherit Q Business data source connections
  • Sharing: Publish apps to your organization's library
  • No Code Required: Business users can build apps without developer help

Creating Your First Q App

There are two ways to create a Q App:

From a Conversation

  1. Chat with Amazon Q Business about a task
  2. When you get a useful result, click "Create Q App"
  3. Q automatically creates input cards for the variable parts
  4. Customize the app layout and save

From Scratch

  1. Navigate to Q Apps in the Q Business console
  2. Click "Create App" and describe what you want
  3. Add input cards (text fields, file uploads, dropdowns)
  4. Add processing cards (Q Business query, text generation)
  5. Add output cards (text display, formatted results)
