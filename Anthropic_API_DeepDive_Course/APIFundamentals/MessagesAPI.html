Messages API and Authentication

The Messages API

The Anthropic Messages API is the primary interface for interacting with Claude models. Every interaction follows a messages-based format with system prompts, user messages, and assistant responses.

Authentication

import anthropic

# API key authentication
client = anthropic.Anthropic(api_key="sk-ant-...")

# Or use environment variable ANTHROPIC_API_KEY
client = anthropic.Anthropic()

Basic API Call

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    system="You are a helpful coding assistant.",
    messages=[
        {"role": "user", "content": "Write a Python function to check if a number is prime."}
    ]
)
print(response.content[0].text)
