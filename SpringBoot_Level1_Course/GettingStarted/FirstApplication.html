Creating Your First Spring Boot Application

Step 1: Generate Project

Visit start.spring.io and configure your project:

  • Project: Maven Project
  • Language: Java
  • Spring Boot Version: Latest stable version
  • Group: com.example
  • Artifact: demo
  • Dependencies: Spring Web

Step 2: Main Application Class

The main class is the entry point of your Spring Boot application:

@SpringBootApplication
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Step 3: Create a Controller

Create a simple REST controller:

@RestController
public class HelloController {
    @GetMapping("/hello")
    public String hello() {
        return "Hello, Spring Boot!";
    }
}

Step 4: Run the Application

Run your application using:

  • IDE: Right-click and run the main class
  • Maven: mvn spring-boot:run
  • Java: java -jar target/demo-0.0.1-SNAPSHOT.jar

Step 5: Test Your Application

Open your browser and navigate to:

http://localhost:8080/hello

You should see "Hello, Spring Boot!" displayed.

Understanding the Magic

Spring Boot automatically:

  • Configured an embedded Tomcat server
  • Set up Spring MVC
  • Configured JSON serialization
  • Enabled component scanning
