GenAI Pricing Models

Token-Based Pricing

Most GenAI services charge per token (roughly 4 characters or 0.75 words). Costs differ between input and output tokens.

ModelInput (per 1M tokens)Output (per 1M tokens)

Claude 3.5 Haiku$0.80$4.00
Claude 3 Opus$15.00$75.00
Titan Text Express$0.20$0.60

Hidden Cost Drivers

  • System prompts: Long system prompts are sent with every request
  • Conversation history: Growing message history increases input tokens
  • RAG context: Retrieved documents add to input token count
  • Retries: Failed requests still consume tokens on the provider side
  • Extended thinking: Thinking tokens billed at output rates

Cost Estimation Formula

monthly_cost = (
    daily_requests *
    avg_input_tokens * input_price_per_token +
    daily_requests *
    avg_output_tokens * output_price_per_token
) * 30
