What Changed: CodeWhisperer to Q Developer

The Evolution

In April 2024, Amazon rebranded CodeWhisperer as part of the broader Amazon Q Developer service. Q Developer includes all CodeWhisperer features plus significant new capabilities.

New Capabilities in Q Developer

  • /dev command: Describe a feature in natural language, Q Developer generates the implementation across multiple files
  • /transform command: Automated Java upgrades (Java 8/11 to Java 17), framework migrations
  • Chat interface: Ask questions about your code, AWS services, and best practices
  • Agent capabilities: Q Developer can read your project structure and make multi-file changes

Migration Steps

  1. Update your IDE plugin to the latest Amazon Q extension
  2. Sign in with the same AWS Builder ID or IAM Identity Center credentials
  3. Your existing customizations and settings carry over automatically
  4. Explore new features: /dev, /transform, and chat
