Understanding MCP

What Is MCP?

The Model Context Protocol (MCP) is an open standard created by Anthropic that defines how AI applications connect to external data sources and tools. Think of it as a USB-C port for AI - a universal connector.

Architecture Overview

  • Host: The AI application (Claude Code, IDE extension, custom app)
  • Client: Maintains a 1:1 connection to a server (managed by the host)
  • Server: Provides tools, resources, and prompts via the MCP protocol
  • Transport: Communication layer (stdio or SSE)

MCP Primitives

PrimitivePurposeExample
ToolsActions the AI can executeQuery database, send email, create file
ResourcesData the AI can readFile contents, database records, API responses
PromptsReusable prompt templatesCode review template, analysis workflow

Why MCP Matters

  • Standardization: Build once, use with any MCP-compatible host
  • Security: Explicit capability declarations, user consent for actions
  • Composability: Connect multiple servers for combined capabilities
