Setup and Configuration

VS Code Installation

  • Open VS Code Extensions panel (Ctrl+Shift+X)
  • Search for "Amazon Q"
  • Install the official Amazon Q extension
  • Sign in with AWS Builder ID (free) or IAM Identity Center (enterprise)

Authentication Options

MethodBest ForFeatures
AWS Builder IDIndividual developersFree tier features
IAM Identity CenterEnterprise teamsPro features + customization

Using Inline Completions

As you type, Amazon Q suggests completions:

  • Tab: Accept the full suggestion
  • Ctrl+Right Arrow: Accept word by word
  • Esc: Dismiss suggestion
  • Alt+C: Manually trigger suggestion

Using the Chat Panel

Open the Q chat panel and ask questions like:

"How do I set up an S3 bucket with versioning in CDK?"
"Explain this Lambda function"
"What's the best way to handle DynamoDB pagination?"
"@workspace How does authentication work in this project?"
