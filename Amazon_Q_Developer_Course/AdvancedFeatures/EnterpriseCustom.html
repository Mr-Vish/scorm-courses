Enterprise Customization

Custom Code References

With Q Developer Pro, you can connect your internal codebase as a reference source. Q will use your company's code patterns, naming conventions, and internal libraries in its suggestions.

Setting Up Customization

  • Upload code to an S3 bucket or connect a CodeStar repository
  • Create a customization resource in the Q Developer console
  • Q indexes your code and uses it for future suggestions
  • Updates automatically when source code changes

Q in AWS Console

Amazon Q is also available directly in the AWS Console for:

  • Troubleshooting: "Why is my Lambda function timing out?"
  • Service guidance: "How do I set up a VPC peering connection?"
  • Cost optimization: "How can I reduce my EC2 costs?"
  • Error diagnosis: Paste CloudWatch errors for instant analysis
