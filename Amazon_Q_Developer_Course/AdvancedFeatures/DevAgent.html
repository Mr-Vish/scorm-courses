The /dev Agent and Code Transformation

The /dev Agent

The /dev command turns Q into an autonomous coding agent that can plan and implement multi-file changes from a natural language description:

/dev Add pagination to the UserController REST API.
Use Spring Data's Pageable interface.
Include page size, page number, and sort parameters.

Q Developer will:

  • Analyze your codebase to understand existing patterns
  • Create a plan for the changes
  • Generate code changes across multiple files
  • Present a diff for your review before applying

Code Transformation

Amazon Q can automatically upgrade your codebase:

  • Java 8/11 to Java 17: Update APIs, patterns, and syntax
  • .NET Framework to .NET Core: Full migration assistance
  • Python 2 to Python 3: Syntax and library updates

Security Scanning

Q Developer scans your code for:

  • OWASP Top 10 vulnerabilities
  • Hard-coded credentials and secrets
  • Insecure cryptographic implementations
  • AWS-specific misconfigurations (open S3 buckets, overly permissive IAM)
