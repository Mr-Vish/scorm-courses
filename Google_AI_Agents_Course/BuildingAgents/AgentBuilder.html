Vertex AI Agent Builder

Vertex AI Agent Builder Overview

Vertex AI Agent Builder is Google Cloud's managed platform for creating AI agents that can use tools, access data stores, and interact with users through natural language.

Agent Architecture

  • Agent: The top-level entity powered by Gemini for reasoning
  • Tools: Functions the agent can invoke (APIs, data stores, code)
  • Data Stores: Structured and unstructured data for grounding
  • Extensions: Pre-built integrations with Google services and third-party APIs
  • Playbooks: Step-by-step instructions for handling specific scenarios

Creating an Agent

from google.cloud import aiplatform
from vertexai.preview import reasoning_engines

# Define tools
def get_weather(city: str) -> dict:
    "Get current weather for a city."
    # Call weather API
    return {"city": city, "temp": 72, "condition": "sunny"}

def search_flights(origin: str, dest: str, date: str) -> list:
    "Search available flights."
    # Call flights API
    return [{"flight": "AA123", "price": 299}]

# Create the agent
agent = reasoning_engines.LangchainAgent(
    model="gemini-1.5-pro",
    tools=[get_weather, search_flights],
    agent_executor_kwargs={"return_intermediate_steps": True}
)

# Deploy to Vertex AI
remote_agent = reasoning_engines.ReasoningEngine.create(
    agent,
    display_name="Travel Assistant Agent"
)
