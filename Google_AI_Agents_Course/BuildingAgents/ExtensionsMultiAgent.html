Custom Extensions with OpenAPI

# Define a custom extension using OpenAPI spec
extension = aiplatform.Extension.create(
    display_name="Inventory API",
    manifest={
        "api_spec": {
            "open_api_gcs_uri": "gs://my-bucket/inventory-api.yaml"
        },
        "auth_config": {
            "auth_type": "API_KEY_CONFIG",
            "api_key_config": {"name": "x-api-key", "api_key_secret": "projects/my-project/secrets/api-key"}
        }
    }
)

Multi-Agent Orchestration

Build systems with specialized agents that collaborate:

  • Router Agent: Classifies user intent and delegates to specialist agents
  • Research Agent: Searches data stores and summarizes findings
  • Action Agent: Executes operations (booking, ordering, updating)
  • Review Agent: Validates actions before execution

LangChain Integration

from langchain_google_vertexai import ChatVertexAI
from langchain.agents import AgentExecutor, create_tool_calling_agent
from langchain_core.prompts import ChatPromptTemplate

llm = ChatVertexAI(model_name="gemini-1.5-pro")

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful travel assistant."),
    ("human", "{input}"),
    ("placeholder", "{agent_scratchpad}")
])

agent = create_tool_calling_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True)
result = executor.invoke({"input": "Find flights from NYC to LA next Friday"})
