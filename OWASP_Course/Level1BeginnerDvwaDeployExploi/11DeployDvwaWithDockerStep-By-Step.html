1.1 Deploy DVWA with Docker (step-by-step)

  1. Open VS Code. Open a new terminal (Ctrl+ ).
  2. Pull and run official DVWA image (community) — one easy command:

docker run --name dvwa -d -p 8080:80 vulnerables/web-dvwa

  • --name dvwa names the container.
  • -p 8080:80 exposes container HTTP port 80 on host 8080.
  • Wait a few seconds, then open http://localhost:8080 in your browser.
  • Login credentials (DVWA default):
  • Username: admin
  • Password: password

If you see errors on first load, DVWA sometimes requires initialization — check container logs:

docker logs -f dvwa

Common errors & fixes

  • docker: command not found → Install Docker Desktop and restart terminal.
  • port is already allocated → change host port: -p 8081:80.
