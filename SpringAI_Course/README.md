# Spring AI Course - SCORM 1.2

## Course Overview
This comprehensive Spring AI course teaches Java developers how to integrate Generative AI capabilities into their Spring Boot applications. The course covers everything from basic concepts to building production-ready AI-powered services.

### Module 1: Introduction to Spring AI
- Understanding Spring AI and its purpose
- Comparing Spring AI with LangChain
- Benefits of using Spring AI for Java developers
- The "Idiomatic Java" advantage

### Module 2: Core Concepts
- Key components: Model, Prompt, Completion, ChatClient
- Understanding the abstraction layer
- Supported AI providers (OpenAI, Google, Azure, Anthropic)
- Spring AI architecture and patterns

### Module 3: Building a Chat Service
- Project setup with Maven dependencies
- Configuration and API key management
- Implementing the AI Controller
- Using the ChatClient fluent API
- Testing and running the application

### Module 4: Advanced Topics
- Retrieval Augmented Generation (RAG)
- Structured Output
- Function Calling
- Prompt Templates
- Streaming Responses
- Best practices and resources

## Learning Objectives
Students will learn to:
- Understand what Spring AI is and why it's valuable for Java developers
- Compare Spring AI with Python-based alternatives like LangChain
- Set up Spring Boot projects with Spring AI dependencies
- Build RESTful AI services using ChatClient
- Integrate with multiple AI providers (OpenAI, Google, Azure, Anthropic)
- Implement advanced features like RAG and Structured Output
- Apply best practices for production AI applications

## Prerequisites
- Solid understanding of Java 8+ features
- Basic familiarity with Spring Boot applications
- Understanding of Maven or Gradle for dependency management
- Knowledge of building RESTful web services
- Basic understanding of HTTP and REST APIs

## Assessment Structure
- **Assessment 1**: Introduction to Spring AI (6 questions, 70% required to pass)
- **Assessment 2**: Core Concepts (6 questions, 70% required to pass)
- **Assessment 3**: Building Chat Service (6 questions, 70% required to pass)
- **Final Assessment**: Comprehensive knowledge check (25 questions, 70% required to pass)

All assessments must be passed with 70% or higher to progress through the course.

1. Upload the entire SpringAI_Course folder to your SCORM-compliant LMS

3. Ensure all file permissions are properly set
4. Test the course in your LMS environment before deployment
