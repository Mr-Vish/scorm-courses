Building a Simple Chat Service

Project Setup with Maven

Let's create a minimal Spring Boot application that uses the ChatClient to generate responses based on a user's message. For this example, we will use the OpenAI model.

Step 1: Maven Dependencies

Add the following to your pom.xml file:

<dependencies> <dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-web</artifactId> </dependency> <dependency> <groupId>org.springframework.ai</groupId> <artifactId>spring-ai-openai-spring-boot-starter</artifactId> </dependency> </dependencies> <dependencyManagement> <dependencies> <dependency> <groupId>org.springframework.ai</groupId> <artifactId>spring-ai-bom</artifactId> <version>1.0.0</version> <type>pom</type> <scope>import</scope> </dependency> </dependencies> </dependencyManagement>

Understanding the Dependencies

  • spring-boot-starter-web: Provides web application capabilities including REST endpoints
  • spring-ai-openai-spring-boot-starter: Auto-configures Spring AI with OpenAI support
  • spring-ai-bom: Bill of Materials that manages Spring AI version compatibility

Configuration

You need to provide your AI provider's API key. Place this in your src/main/resources/application.properties file:

# Replace with your actual OpenAI API Key spring.ai.openai.api-key=<YOUR_OPENAI_API_KEY>

Getting an API Key

To obtain an OpenAI API key:

  • Visit platform.openai.com
  • Create an account or sign in
  • Navigate to API Keys section
  • Generate a new API key
  • Copy and paste it into your application.properties

Project Structure

Your Spring Boot project should have the following structure:

src/ ├── main/ │ ├── java/ │ │ └── com/example/aidemo/ │ │ ├── AiDemoApplication.java │ │ └── AiController.java │ └── resources/ │ └── application.properties └── test/ └── java/

Spring Boot Application Class

Create a standard Spring Boot application class:

package com.example.aidemo; import org.springframework.boot.SpringApplication; import org.springframework.boot.autoconfigure.SpringBootApplication; @SpringBootApplication public class AiDemoApplication { public static void main(String[] args) { SpringApplication.run(AiDemoApplication.class, args); } }

What Happens Behind the Scenes

When you add the Spring AI starter dependency:

  • Spring Boot auto-configures the ChatClient bean
  • The OpenAI client is initialized with your API key
  • All necessary HTTP clients and serializers are set up
  • The ChatClient is ready to be injected into your components
