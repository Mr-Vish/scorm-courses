Advanced Agent Features

Return of Control

Instead of calling a Lambda, the agent can return control to your application with the action details, letting you execute the action and return results. This is useful for actions requiring human approval.

Multi-Agent Collaboration

Bedrock supports supervisor agents that coordinate multiple sub-agents:

  • Supervisor: Routes requests to the appropriate specialist agent
  • Sub-agents: Specialized agents (order handling, product info, etc.)
  • Shared context: Agents can pass context between each other

Monitoring and Tracing

Enable trace logging to see the agent's reasoning process:

  • Pre-processing: Input analysis and intent detection
  • Orchestration: Planning which actions to take
  • Post-processing: Formatting the final response
