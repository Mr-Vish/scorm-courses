Introduction to Firebase Genkit

What Is Genkit?

Firebase Genkit is an open-source framework from Google for building AI-powered applications. It provides a unified interface for working with multiple AI models, structured generation, retrieval, and evaluation.

Key Concepts

  • Flows: Type-safe, observable functions that can call AI models and tools
  • Models: Abstraction over LLM providers (Gemini, GPT, Claude, Ollama)
  • Prompts: Managed prompt templates with input/output schemas
  • Retrievers: Built-in RAG support with pluggable vector stores
  • Evaluators: Test AI output quality with automated metrics

Setup

npm install genkit @genkit-ai/googleai

// Initialize Genkit
import { genkit } from 'genkit';
import { googleAI, gemini15Flash } from '@genkit-ai/googleai';

const ai = genkit({
  plugins: [googleAI()],
  model: gemini15Flash,
});

Genkit Developer UI

Run npx genkit start to launch a visual developer UI for testing flows, inspecting traces, and evaluating outputs.
