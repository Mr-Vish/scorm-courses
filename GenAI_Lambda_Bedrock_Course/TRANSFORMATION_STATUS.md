### Module 1: Serverless GenAI Foundations

### Module 2: Building GenAI Applications
4. API Integration - REST APIs, WebSocket streaming, request/response handling
5. State Management - DynamoDB conversation history, RAG with vector databases
6. Advanced Patterns - Long-running tasks, Step Functions, batch processing

9. Deployment and CI/CD - SAM, CDK, automated testing, blue/green deployments
