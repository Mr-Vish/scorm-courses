Setting Up Data Connectors

Connecting Data Sources

Each data connector syncs documents from an external system into the Q Business index:

  • S3 Connector: Ingest PDF, DOCX, HTML, TXT, CSV files from S3 buckets
  • Confluence Connector: Sync wiki spaces, pages, and blog posts
  • SharePoint Connector: Index documents, lists, and pages
  • Web Crawler: Crawl and index public or internal web pages

Sync Modes

ModeBehaviorUse Case
Full syncRe-index all documentsInitial setup, major changes
Incremental syncOnly new/modified documentsRegular updates (scheduled)

Access Control Lists (ACLs)

Q Business can inherit permissions from source systems. When a user asks a question, Q only retrieves documents that user has access to in the source system. This requires:

  • IAM Identity Center integration for user mapping
  • Source system credentials with permission-reading access
  • ACL-aware connector configuration
