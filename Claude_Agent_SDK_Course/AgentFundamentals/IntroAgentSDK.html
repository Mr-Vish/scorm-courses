Introduction to the Claude Agent SDK

What Is the Agent SDK?

The Claude Agent SDK is Anthropic's framework for building autonomous AI agents that can reason, use tools, and complete multi-step tasks. It provides the scaffolding to go from a simple API call to a fully functional agent.

Agent vs Chatbot

FeatureChatbotAgent
InteractionSingle turn Q&AMulti-step autonomous execution
ToolsNone or limitedMultiple tools orchestrated dynamically
PlanningReactivePlans, executes, and adapts
MemoryConversation onlyState persistence across steps

Core Components

  • Agent Loop: The reasoning cycle: observe, plan, act, reflect
  • Tool Definitions: Functions the agent can invoke
  • System Prompt: Instructions defining agent behavior and constraints
  • Guardrails: Safety checks on inputs, outputs, and tool calls
