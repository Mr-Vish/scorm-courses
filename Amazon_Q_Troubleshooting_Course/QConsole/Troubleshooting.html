Cost Analysis

Q can help you understand and optimize your AWS spending:

User: "What's driving my high EC2 costs this month?"
Q: "Your EC2 costs increased 45% this month. Key contributors:
    - 3 m5.4xlarge instances running 24/7 ($1,847/mo)
    - 2 p3.2xlarge GPU instances ($7,344/mo)
    - Consider:
      1. Reserved Instances for steady workloads (save ~40%)
      2. Spot for the GPU instances if interruptible
      3. Right-sizing the m5.4xlarge if CPU util < 40%"
