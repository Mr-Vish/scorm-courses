Chain-of-Thought and Advanced Patterns

Chain-of-Thought (CoT) Prompting

Ask the model to reason step-by-step before giving a final answer. This significantly improves accuracy on math, logic, and multi-step reasoning tasks:

# Without CoT
Q: If a store has 3 shelves with 8 books each, and 5 books are sold, how many remain?
A: 19  (correct, but no reasoning shown)

# With CoT
Q: If a store has 3 shelves with 8 books each, and 5 books are sold, how many remain?
Think step by step.
A: Let me work through this:
   1. Total books: 3 shelves x 8 books = 24 books
   2. Books sold: 5
   3. Remaining: 24 - 5 = 19 books
   The answer is 19.

Advanced Prompting Patterns

PatternDescriptionUse Case
Self-consistencyGenerate multiple CoT answers, pick the majorityMath, logic problems
Tree of ThoughtExplore multiple reasoning paths and evaluate eachComplex planning
ReActInterleave reasoning and action (tool use)Agent-based tasks
Least-to-MostDecompose into sub-problems, solve smallest firstMulti-step tasks
Role promptingAssign an expert role to the modelDomain-specific tasks

Combining Patterns

# Role + Few-shot + CoT combined
You are a senior database administrator.

Example: "Our query takes 30 seconds"
Analysis:
1. Check EXPLAIN plan for full table scans
2. Verify indexes exist on WHERE clause columns
3. Check for N+1 query patterns
Recommendation: Add composite index on (user_id, created_at)

Now analyze: "Our dashboard page loads slowly when filtering by date range"

When CoT Helps vs Hurts

  • Helps: Math, logic, multi-step reasoning, code debugging, complex analysis
  • Hurts/Neutral: Simple classification, translation, creative writing, factual recall
  • Overhead: CoT uses more output tokens, increasing cost and latency
