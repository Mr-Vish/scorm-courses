Gemini Nano and Android Integration

On-Device AI with Gemini Nano

Gemini Nano is Google's smallest model, designed to run directly on mobile devices. It enables AI features without network connectivity and with minimal latency.

Supported Devices and Requirements

  • Pixel 8 Pro and newer: Native Gemini Nano support via AICore
  • Samsung Galaxy S24+: Available through Samsung's AI integration
  • Android 14+: Required for AICore system service
  • Minimum RAM: 6GB+ recommended for smooth inference

Setting Up Gemini Nano in Android

// build.gradle.kts
dependencies {
    implementation("com.google.ai.edge.litert:litert:1.0.0")
    implementation("com.google.android.gms:play-services-aicore:16.0.0")
}

// Check availability
val aiCoreManager = AICoreManager.getInstance(context)
val isAvailable = aiCoreManager.isFeatureAvailable(Feature.TEXT_GENERATION)

On-Device Text Generation

// Generate text on-device
val generationConfig = GenerationConfig.builder()
    .setMaxOutputTokens(256)
    .setTemperature(0.7f)
    .build()

val model = GenerativeModel(
    modelName = "gemini-nano",
    generationConfig = generationConfig
)

val response = model.generateContent("Summarize this email: $emailText")
val summary = response.text
