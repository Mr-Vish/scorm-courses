Tips for Better Suggestions

  • Good variable names: Descriptive names give Gemini more context
  • Type annotations: Types help Gemini understand intended data shapes
  • Comments before code: Natural language comments guide generation
  • Open related files: Gemini uses open files as context

Security Considerations

Always review AI-generated code for security issues. Common pitfalls include SQL injection in generated queries, missing input validation, and hardcoded credentials in examples.
