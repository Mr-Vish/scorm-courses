Image Alt Text and Video Captioning

AI for Digital Accessibility

Over one billion people worldwide live with some form of disability. GenAI can dramatically improve digital accessibility by automatically generating alt text for images, captions for video, document remediation, and assistive tools - making content accessible to everyone.

Automated Alt Text Generation

Vision-language models can describe images for screen reader users:

from openai import OpenAI
import base64

client = OpenAI()

def generate_alt_text(image_path: str, context: str = "") -> str:
    '''Generate descriptive alt text for an image.'''
    with open(image_path, "rb") as f:
        image_data = base64.b64encode(f.read()).decode("utf-8")

    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{
            "role": "user",
            "content": [
                {"type": "text", "text": (
                    "Generate concise, descriptive alt text for this image. "
                    "Follow WCAG 2.1 guidelines:
"
                    "- Describe the content and function, not just appearance
"
                    "- Keep it under 125 characters when possible
"
                    "- Do not start with 'Image of' or 'Picture of'
"
                    f"Page context: {context}"
                )},
                {"type": "image_url", "image_url": {
                    "url": f"data:image/png;base64,{image_data}",
                }},
            ],
        }],
    )
    return response.choices[0].message.content

alt = generate_alt_text("chart.png", context="Quarterly revenue report page")
# Output: "Bar chart showing Q1-Q4 revenue: Q1 $2.1M, Q2 $2.8M, Q3 $3.5M, Q4 $4.2M"

Alt Text Quality Guidelines

Image TypeWhat to DescribeExample
Informational photoSubject, action, context"Engineer testing circuit board with a multimeter"
Chart / GraphType, data, trend, key values"Line chart showing 15% growth in monthly active users from Jan to Dec"
Decorative imageMark as decorative (empty alt)alt="" (screen readers skip it)
Functional (button/link)The function, not the appearance"Search" not "Magnifying glass icon"
Complex diagramSummary + long descriptionBrief alt + link to detailed text description

Video Captioning

from openai import OpenAI

client = OpenAI()

def transcribe_and_caption(audio_path: str) -> dict:
    '''Transcribe audio and generate timed captions.'''
    # Step 1: Transcribe with Whisper
    with open(audio_path, "rb") as f:
        transcript = client.audio.transcriptions.create(
            model="whisper-1",
            file=f,
            response_format="verbose_json",
            timestamp_granularities=["segment"],
        )

    # Step 2: Format as WebVTT captions
    vtt_lines = ["WEBVTT
"]
    for segment in transcript.segments:
        start = format_timestamp(segment["start"])
        end = format_timestamp(segment["end"])
        vtt_lines.append(f"{start} --> {end}")
        vtt_lines.append(segment["text"].strip())
        vtt_lines.append("")

    return {
        "full_text": transcript.text,
        "vtt": "
".join(vtt_lines),
        "segments": transcript.segments,
    }

def format_timestamp(seconds: float) -> str:
    h = int(seconds // 3600)
    m = int((seconds % 3600) // 60)
    s = int(seconds % 60)
    ms = int((seconds % 1) * 1000)
    return f"{h:02d}:{m:02d}:{s:02d}.{ms:03d}"

Accessibility Benefits

  • Screen reader users: Alt text makes images accessible to blind and low-vision users
  • Deaf and hard of hearing: Captions and transcripts make audio/video content accessible
  • Cognitive disabilities: AI-generated summaries and simplified text help comprehension
  • Motor disabilities: Voice-controlled AI interfaces reduce need for keyboard/mouse input
