Prefilling and Output Control

Assistant Prefilling

When using the API, you can prefill the beginning of Claude's response to steer its output format:

# API call with prefilled assistant response
messages = [
    {"role": "user", "content": "List the top 3 issues"},
    {"role": "assistant", "content": "1."}
]
# Claude will continue from "1." in a numbered list format

Forcing JSON Output

messages = [
    {"role": "user", "content": "Extract entities from: ..."},
    {"role": "assistant", "content": "{"}
]
# Claude will produce valid JSON starting with {

Output Constraints

  • Length control: "Respond in exactly 3 bullet points"
  • Format enforcement: "Reply only with valid JSON, no explanation"
  • Language restriction: "Use only Python 3.12+ syntax"
  • Scope limitation: "Only discuss the authentication module"

Stop Sequences

Configure stop sequences in the API to make Claude halt generation at specific tokens. Useful for structured extraction where you want Claude to fill in a template.
