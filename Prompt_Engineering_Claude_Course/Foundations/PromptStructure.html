Prompt Structure and Anatomy

The Anatomy of an Effective Prompt

Every prompt to Claude consists of key structural elements that determine the quality of the response:

  • System Prompt: Sets Claude's role, behavior, and constraints
  • User Message: The actual request or question
  • Context: Background information, examples, or data
  • Output Format: Desired structure of the response

System Prompts

The system prompt is the most powerful control mechanism. It defines who Claude is for the conversation:

You are a senior Java developer with 15 years of experience
in Spring Boot microservices. You review code for:
- Performance issues
- Security vulnerabilities
- SOLID principle violations
Always provide specific line references and suggest fixes.

Key Principles

PrincipleDescriptionExample
Be SpecificVague prompts get vague answers"List 5 bugs" vs "Find bugs"
Provide ContextGive Claude the information it needsInclude relevant code, docs, constraints
Define FormatTell Claude how to structure output"Respond in JSON" or "Use bullet points"
Set ConstraintsBound the response scope"Under 200 words" or "Only Java solutions"
