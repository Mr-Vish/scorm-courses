Human-in-the-Loop

LangGraph supports interrupting graph execution for human approval:

# Compile with interrupt points
app = graph.compile(
    checkpointer=memory_checkpointer,
    interrupt_before=["execute_action"]  # Pause before this node
)

# Run until interrupt
result = app.invoke({"messages": [user_query]})

# Human reviews and approves
app.update_state(thread_id, {"approved": True})

# Resume execution
result = app.invoke(None, config={"thread_id": thread_id})
