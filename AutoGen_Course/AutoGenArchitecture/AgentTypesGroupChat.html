Agent Types and Group Chat

What is AutoGen?

AutoGen is Microsoft's open-source framework for building multi-agent systems where multiple AI agents collaborate through conversations. Agents can plan, write code, execute it, debate results, and iteratively solve complex problems together.

Agent Types

Agent TypeDescriptionUse Case
AssistantAgentLLM-powered agent that generates responses and codePlanning, analysis, code generation
UserProxyAgentRepresents the human user, can execute codeCode execution, human feedback
ConversableAgentBase class for custom agents with configurable behaviorSpecialized roles and custom logic
GroupChatManagerOrchestrates multi-agent conversationsManaging turn-taking and flow

Two-Agent Conversation

from autogen import AssistantAgent, UserProxyAgent

# Configuration for the LLM
llm_config = {
    "model": "gpt-4o",
    "api_key": os.environ["OPENAI_API_KEY"],
}

# Create an assistant agent
assistant = AssistantAgent(
    name="data_analyst",
    system_message="You are a data analyst. Write Python code to analyze data.",
    llm_config=llm_config,
)

# Create a user proxy that can execute code
user_proxy = UserProxyAgent(
    name="user",
    human_input_mode="NEVER",  # Fully autonomous
    code_execution_config={"work_dir": "workspace"},
)

# Start the conversation
user_proxy.initiate_chat(
    assistant,
    message="Analyze the sales data in sales.csv and create a trend chart."
)

Group Chat

Group chat enables three or more agents to collaborate on a task:

from autogen import GroupChat, GroupChatManager

# Define specialized agents
planner = AssistantAgent(name="planner", system_message="You plan tasks.", llm_config=llm_config)
coder = AssistantAgent(name="coder", system_message="You write code.", llm_config=llm_config)
reviewer = AssistantAgent(name="reviewer", system_message="You review code.", llm_config=llm_config)

# Create group chat
group_chat = GroupChat(
    agents=[user_proxy, planner, coder, reviewer],
    messages=[],
    max_round=12,
    speaker_selection_method="auto",  # LLM decides who speaks next
)

manager = GroupChatManager(groupchat=group_chat, llm_config=llm_config)
user_proxy.initiate_chat(manager, message="Build a REST API for user management.")

Speaker Selection Methods

  • auto: The LLM decides which agent should speak next based on conversation context
  • round_robin: Agents take turns in a fixed order
  • random: A random agent is selected each turn
  • Custom function: Provide your own selection logic for deterministic workflows
