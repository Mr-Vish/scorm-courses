Core Slash Commands

Built-in Commands

Claude Code provides several slash commands for common operations:

CommandPurpose
/helpShow available commands and usage tips
/clearClear conversation context and start fresh
/compactCompact the conversation to save context window
/costShow token usage and cost for current session
/doctorDiagnose common issues with your setup
/initInitialize a CLAUDE.md project guide
/reviewReview code changes (works with git)
/terminal-setupInstall shell integration for better terminal support

The CLAUDE.md File

Running /init creates a CLAUDE.md file in your project root. This file acts as a persistent instruction set for Claude Code, containing:

  • Project description and tech stack
  • Coding conventions and style guides
  • Build and test commands
  • Common workflows and patterns

Keyboard Shortcuts

  • Escape: Cancel current generation
  • Tab: Accept autocomplete suggestions
  • Ctrl+C: Interrupt current operation
  • Up/Down arrows: Navigate command history
