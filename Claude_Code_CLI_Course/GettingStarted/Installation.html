Installation and Configuration

Prerequisites

  • Node.js 18+ installed on your system
  • npm (comes with Node.js)
  • An Anthropic API key or Claude Pro/Max subscription

Installation

Install Claude Code globally via npm:

npm install -g @anthropic-ai/claude-code

First Launch

Navigate to your project directory and run:

cd your-project
claude

On first launch, Claude Code will prompt you to authenticate. You can use your Anthropic API key or sign in with your Claude account.

Configuration Options

Claude Code stores settings in ~/.claude/. Key configuration files include:

  • settings.json: Global preferences, model selection, permissions
  • keybindings.json: Custom keyboard shortcuts
  • .claude/settings.json (per-project): Project-specific configuration

Verifying Installation

claude --version
claude --help
