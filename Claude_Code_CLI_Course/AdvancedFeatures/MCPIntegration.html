MCP Server Integration

Model Context Protocol (MCP)

Claude Code supports MCP servers that extend its capabilities with custom tools and data sources. MCP is an open protocol that lets you connect Claude to external systems.

Adding MCP Servers

Configure MCP servers in your settings:

{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "your-token"
      }
    },
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": {
        "DATABASE_URL": "postgresql://..."
      }
    }
  }
}

Popular MCP Servers

  • GitHub: Create PRs, manage issues, search code across repos
  • PostgreSQL/MySQL: Query databases, inspect schemas
  • Filesystem: Extended file operations with permissions
  • Brave Search: Web search capabilities
  • Puppeteer: Browser automation and web scraping

Building Custom MCP Servers

You can build your own MCP servers using the SDK to expose any API, database, or service to Claude Code. Servers communicate via stdio or SSE transport.
